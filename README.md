My i3 configuration.

- ttf-font-awesome. Icon fonts.
- rxvt-unicode. Terminal.
- oh-my-zsh
- zsh
- powerlevel9k zsh theme
- mononoki fonts
- powerline fonts
- Consolas for Powerline fonts
- Network Manager and Network Manager Applet
- Nitrogen for Wallpaper
